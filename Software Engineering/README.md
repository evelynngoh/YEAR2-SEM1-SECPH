Software Engineering course provided valuable understanding of software development processes, software architecture, object-oriented analysis and design, and software testing techniques used in real-world software engineering practices. Throughout the course, knowledge of software process models such as Agile, Waterfall, Spiral, and Rational Unified Process (RUP) was significantly strengthened through lectures, case studies, assignments, and project-based learning activities. Practical exposure to UML modelling, requirements engineering, architectural design, object-oriented design, and software verification and validation enhanced analytical thinking, problem-solving, and documentation skills while improving the ability to produce professional software artifacts such as SRS, SDD, and testing documents. In addition, the industry-based software engineering project strengthened teamwork, leadership, communication, and project management skills through collaborative discussions, presentations, peer evaluations, and industry engagement activities. The course also increased understanding of how structured software engineering methodologies contribute to the development of high-quality and maintainable software systems. Overall, this course successfully enhanced both theoretical understanding and practical competency in software engineering concepts, system modelling, and software development practices.
