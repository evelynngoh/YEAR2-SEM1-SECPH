Database course provided valuable understanding and practical experience in database systems, database design, and database application development for real-world problems. Throughout the course, knowledge of database concepts, relational models, SQL programming, relational algebra, normalization, and entity relationship diagrams (ERD) was significantly strengthened through lectures, laboratory work, assignments, and project-based learning activities. Practical exposure to MySQL and database management system (DBMS) tools enhanced technical skills in constructing SQL queries, designing conceptual and logical databases, and developing database application solutions. In addition, the industry-based group project improved teamwork, communication, analytical thinking, and problem-solving skills through collaboration with industry representatives during the system analysis, design, implementation, and testing phases. The course also increased understanding of how structured database design and efficient data management support organizational operations and information systems in real-world environments. Overall, this course successfully enhanced both theoretical understanding and practical competency in database design, SQL programming, and database application development.
