Network Communications course provided valuable understanding of computer networking concepts, data communication principles, and Internet protocols based on the TCP/IP protocol stack. Throughout the course, knowledge of application, transport, network, link, and wireless network layers was significantly strengthened through lectures, laboratory exercises, tests, and project-based learning activities. Practical exposure to networking tools such as Wireshark and Cisco Packet Tracer enhanced technical skills in network analysis, packet inspection, subnetting, routing, and local area network (LAN) design. In addition, the group project improved teamwork, analytical thinking, problem-solving, and technical report writing skills through the design and implementation of a small network solution based on a real-world case study. The course also increased understanding of how network communication technologies support Internet services, wireless communication, and modern computing environments. Overall, this course successfully enhanced both theoretical understanding and practical competency in computer networking and data communication technologies.
